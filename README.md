# Data Analytics Internship Projects - DecodeLabs

## 📌 Overview

This repository contains the solutions for all three Data Analytics internship projects assigned by DecodeLabs.

The projects cover the complete Data Analytics workflow:

1. Data Cleaning & Preparation
2. Exploratory Data Analysis (EDA)
3. SQL Data Analysis

Each task is organized in a separate folder with its corresponding datasets, notebooks, SQL scripts, and analysis files.

# 🚀 How to Run

## Clone Repository

```bash
git clone https://github.com/Mahendra-Palsaniya/DecodeLabs-Intern.git
```

## Open Jupyter Notebook

```bash
jupyter notebook
```

## Run Task 1

Open:

```text
Task-1/cleaning-task-1.ipynb
```

## Run Task 2

Open:

```text
Task-2/EDA-task-2.ipynb
```

## Run Task 3

Execute SQL queries from:

```text
Task-3/SQL_Queries.sql
```

against:

```text
Task-3/decodedb.db
```
